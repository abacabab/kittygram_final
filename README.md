# Kittygram

Этот репозиторий содержит код приложения, в котором можно делиться своими котами и кошками, добавляя записи о них, содержащие: имя, цвет, год рождения, достижения, фото.

## Технологический стек

**Backend:**
* Django - основная платформа
* PostgreSQL - база данных

**Frontend:**
* React 18 - пользовательский интерфейс;

**DevOps:**
* Docker - контейнеризация;
* GitHub Actions - CI/CD;
* Nginx - прокси-сервер.

## Как запустить проект:

Клонировать репозиторий и перейти в него в командной строке:
```bash
git clone https://github.com/abacabab/kittygram-final-ad.git
```
```bash
cd kittygram-final-ad
```
Создать файл .env и заполнить его следующими переменными:

POSTGRES_DB
POSTGRES_USER - имя пользователя базой данных
POSTGRES_PASSWORD - пароль пользователя базой данных
DB_NAME - название базы данных
SECRET_KEY - Ключ. Рекомендуется генерировать надёжный случайный ключ
ALLOWED_HOSTS - список доменных имён или IP-адресов, для которых Django-проект может обслуживать запросы

В репозитории выполнить команду:
```bash
docker compose up --build
```
